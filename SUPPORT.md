# Getting help

Thanks for using Domestique! Here's where to go:

- **How-to & setup** — start with the [README](./README.md) and the per-tool
  [recipes](./docs/recipes/) (Claude Code, Codex, Cursor, aider, and any
  OpenAI/Anthropic SDK).
- **Questions & ideas** — open a [GitHub Discussion](https://github.com/domestique-labs/domestique-oss/discussions).
- **Bugs & feature requests** — open an [issue](https://github.com/domestique-labs/domestique-oss/issues/new/choose)
  using the templates.
- **Security vulnerabilities** — do **not** open a public issue; report privately via
  [GitHub Private Vulnerability Reporting](https://github.com/domestique-labs/domestique-oss/security/advisories/new)
  (see [SECURITY.md](./SECURITY.md)).

Please search existing issues and discussions before opening a new one.
